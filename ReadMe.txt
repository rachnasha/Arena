The directory structure is as below
[rsha@novusnycmc229 ~/practice/Artist]$ ls
ReadMe.txt	build.sbt	project		src		target

src:

     main:
        contains code, to read from file, process the file and write to file
        ArtistFileService.scala : Reads data from file, calls the counter to find artist for the given count and
        writes to the output file.
        Apps Assumes that file names/paths are valid and no checks are done on the input and output file path/names/extensions.
        ArtistPairsCounter.scala: The file that holds logic to find pairs with a certain count from the lines in the file.

     resources:
         directory that has the test input file ( provided) and sample output files (generated by app).

     test:
         Contains test file that test ArtistPairsCounter logic with smaller inputs.


build.sbt:
      build file for the project
